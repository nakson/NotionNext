# 🎮 像素风主题 - 快速开始指南

## ⚡ 5分钟快速体验

### 1️⃣ 启用主题

在项目根目录的 `blog.config.js` 中确保设置:

```javascript
THEME: 'custom'
```

### 2️⃣ 运行开发服务器

```bash
npm run dev
```

访问 `http://localhost:3000` 查看效果

### 3️⃣ 查看效果

你会看到:

- 🌙 深蓝黑背景
- 💫 霓虹青色边框和文本
- 🎬 卡片上的扫描线动画
- 🎮 游戏风格的按钮和菜单

## 🎯 自定义主题色

想改成其他霓虹色? 编辑 `themes/custom/config.js`:

```javascript
// 改这两行:
HEXO_THEME_COLOR: '#00ffff',        // 改成你喜欢的颜色
HEXO_THEME_NEON_ACCENT: '#ff00ff',  // 改成副色
```

**颜色建议**:

- 🟢 任天堂绿: `#00ff00`
- 🔴 红白机红: `#ff0000`
- 🟠 复古橙: `#ff8800`
- 🩵 浅蓝: `#00ff88`

## 🔧 常用定制

### 改变背景图

1. 准备一张 gif 或 png 图片，放在 `public/images/`
2. 编辑 `config.js`:

```javascript
HEXO_THEME_BG_PIXEL_ART_URL: '/images/your-background.gif',
HEXO_THEME_BG_ART_OPACITY: 0.15  // 调整透明度 0-1
```

### 改变字体大小

在 `style.js` 中找到对应的 `font-size` 修改

### 添加新的按钮/组件

参考 [PIXEL_DESIGN_GUIDE.md](./PIXEL_DESIGN_GUIDE.md) 中的"添加新组件时"部分

## 🐛 常见问题

### Q: 颜色不对劲?

**A:** 清除浏览器缓存或用无痕模式，确保 CSS 变量正确加载

### Q: 字体显示不出来?

**A:** 检查 Google Fonts 是否可访问，浏览器控制台看是否有网络错误

### Q: 手机上看不清楚?

**A:** 这是正常的! 像素风在小屏幕上有挑战，可在 `style.js` 调整手机端的字体大小

### Q: 想改成其他风格怎么办?

**A:** 这个主题已完全像素风化，如需现代风格参考其他主题配置
