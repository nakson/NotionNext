## 🎨 核心设计特性

### 色彩系统

```
背景:    #0f172a (深蓝黑)
卡片:    #1e293b (暗蓝灰)
主色:    #00ffff (霓虹青) ← 边框/强调
副色:    #ff00ff (霓虹紫) ← 悬浮/阴影
文本:    #e0e0e0 (浅灰)
```

### 交互效果

**悬浮状态 (Hover)**:

```
- 颜色切换: 青 → 紫 或 文本 → 反色
- 变换: translate(-1px, -1px) 向左上微移
- 阴影: 增大偏移 3-4px
- 过渡: 100ms (快速，保持像素感)
```

**按下状态 (Active)**:

```
- 变换: translate(2px, 2px) 向右下压入
- 阴影: 缩小或消失
- 反馈: 物理感十足
```

### 特殊效果

🔹 **扫描线动画**

- 卡片上的霓虹线条从上到下扫过 (3秒循环)
- 增强 CRT 显示器复古感

🔹 **背景纹理**

- 页面背景有细微扫描线纹理
- opacity: 0.03，不影响可读性

🔹 **像素化渲染**

- `image-rendering: pixelated` 保证图片边缘锐利
- 所有图表/图表都会呈现 8-bit 风格

🔹 **自定义元素**

- 滚动条: 霓虹青边框 + 悬浮时变紫
- 输入框: 像素风边框 + 焦点时变紫
- 表格: 完整边框 + 表头霓虹青背景
- 代码块: 霓虹青边框 + 阴影

所有样式均使用 CSS 变量，易于维护和扩展:

```css

--pixel-secondary      /* 霓虹紫 */
--pixel-warning        /* 霓虹黄 */
--pixel-bg            /* 背景深蓝黑 */

--pixel-text          /* 浅灰文本 */
--pixel-text-muted    /* 中灰文本 */
--font-pixel          /* 像素字体 */
--font-pixel-title    /* 像素标题字体 */
```
